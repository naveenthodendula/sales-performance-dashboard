# 📊 Sales Performance Dashboard (Excel)

## 📌 Project Overview
This project focuses on analyzing sales data using Microsoft Excel to uncover key business insights.  
An interactive dashboard was created to track performance metrics such as revenue, product sales, and monthly trends.  
The goal is to help businesses make data-driven decisions by visualizing important KPIs.

---

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers  
- Data Cleaning Techniques  

---

## 📊 Key Features
- Interactive dashboard for easy data exploration  
- Monthly sales trend analysis  
- Product-wise performance tracking  
- Revenue breakdown visualization  

---

## 🔍 Key Insights
- Identified top-performing products contributing the highest revenue  
- Observed monthly sales trends and seasonal patterns  
- Highlighted underperforming products needing improvement  
- Provided clear visual insights for better business decision-making
